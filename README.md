## Description

A microservice for managing services you provide across different regions

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# watch mode
$ yarn start:dev

# production mode
$ yarn build
$ yarn start:prod

# docker image
$ docker run -t -p 8080:8080 vogue:1.0.0
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
